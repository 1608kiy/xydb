# 02 功能对齐清单（WEB -> Mobile）

## 1. 页面级对齐矩阵

| WEB 页面 | 移动端页面 | 首发优先级 | 说明 |
|---|---|---|---|
| 登录页面.html | 登录页 | P0 | 支持手机号/邮箱、管理员识别逻辑 |
| 注册页面.html | 注册页 | P0 | 与 WEB 校验规则一致 |
| 待办页面.html | 待办页 | P0 | 任务列表、筛选、增删改查、子任务 |
| 日历页面.html | 日历页 | P0 | 月/周/日切换、任务排期展示 |
| 番茄钟页面.html | 番茄页 | P0 | 模式切换、计时、记录入库 |
| 数据周报页面.html | 周报页 | P0 | 趋势图、分类图、汇总指标 |
| 打卡页面.html | 打卡页 | P0 | 打卡、最近记录、日历标记 |
| 个人中心页面.html | 我的页 | P0 | 用户资料、退出登录、基础设置 |
| 后台管理页面.html | 管理页（可选） | P1 | 建议首发先仅 WEB 保留 |
| 官网.html | 官网（H5/外链） | P2 | 可直接复用 WEB 页面 |

## 2. 功能项拆解

### 2.1 认证与账号

- 登录：账号+密码。
- 注册：昵称+邮箱/手机+密码。
- 找回密码。
- 获取当前用户信息。
- 退出登录。

### 2.2 任务域

- 任务列表查询。
- 新建任务。
- 编辑任务（标题、描述、状态、优先级、时间）。
- 删除任务。
- 子任务更新。
- 标签列表、创建标签。

### 2.3 专注与打卡

- 番茄记录创建。
- 番茄记录查询。
- 打卡创建。
- 打卡日历查询。
- 最近打卡查询。

### 2.4 报表

- 总览数据。
- 每日趋势。
- 分类统计。

### 2.5 管理能力（建议分阶段）

- 用户列表查询。
- 删除用户/批量删除。
- 创建管理员。

## 3. 一致性验收清单

- 字段一致：任务 status、priority、tags 与 WEB 一致。
- 行为一致：删除、编辑、完成任务后，跨端视图一致。
- 统计一致：周报数据同一账号跨端一致。
- 管理一致：管理员账号识别逻辑一致（`admin` 与管理邮箱）。

## 4. 首发不做（明确边界）

- 不做多租户与组织协作复杂权限。
- 不做离线编辑冲突自动合并高级策略。
- 不做聊天 AI 的复杂多轮上下文持久化。
