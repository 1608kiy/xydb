## 5. 协作修改推荐流程
1. 本地先改并自测。
2. 使用 git 提交并推送到 GitHub。
3. 远程登录服务器后拉取或用 scp 上传变更文件。
4. 前端变更：刷新页面并检查控制台报错。
5. 后端变更：重启服务并检查健康接口。
