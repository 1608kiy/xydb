### 一、产品概述

- **产品名称**：轻悦待办  
- **产品类型**：个人效率管理 Web 应用  
- **主要目标**：通过待办、日历、番茄钟、数据周报、打卡、个人中心等功能，帮助用户管理任务、规划时间、养成习惯并复盘效率。  
- **终端形态**：响应式 H5（桌面 + 移动端），多个独立 HTML 页面，通过底部 Tab 导航和登录入口进行串联。

---

### 二、整体信息架构

- **登录页**：`登录页面.html`
- **功能主页（底部 Tab 六个）**：
  1. 待办清单页：`待办页面.html`
  2. 日历视图页：`日历页面.html`
  3. 番茄专注页：`番茄钟页面.html`
  4. 数据周报页：`数据周报页面.html`
  5. 打卡页：`打卡页面.html`
  6. 个人中心页：`个人中心页面.html`

- **导航规则**：
  - 登录成功后默认进入待办清单页。
  - 底部 Tab 在所有功能页固定显示，点击后跳转到对应页面。
  - 顶部返回按钮通常跳回待办页（或预留到“应用列表/首页”）。

---

### 三、页面级需求

#### 3.1 登录页面（`登录页面.html`）

- **核心目标**：提供账号登录 / 注册入口。  
- **功能要点**：
  - 登录 / 注册 Tab 切换。
  - 登录表单：
    - 账号：手机号/邮箱。
    - 密码输入 + 显示/隐藏切换。
    - “记住我”复选框。
    - “忘记密码”链接（暂为占位）。
    - 登录按钮：本版本中前端校验非空后，直接跳转到 `待办页面.html`。
  - 注册表单：
    - 昵称、手机号/邮箱、密码、确认密码。
    - 密码强度检测（弱/中/强）+ 强度条。
    - 两次密码一致性校验。
    - 同意用户协议与隐私政策勾选。
    - 注册按钮（当前模拟注册成功，切回登录 tab）。
  - 第三方登录按钮：微信、苹果、谷歌、访客模式（当前为前端提示，未接第三方 SDK）。
  - 底部版权与协议链接：隐私政策、用户协议、帮助中心（占位）。

---

#### 3.2 待办清单页（`待办页面.html`）

- **核心目标**：管理日常任务、查看分组任务列表和任务详情。  
- **主要模块**：

1. **顶部导航**
   - 返回按钮（回到待办或上级应用）。
   - 页面标题“待办清单”。
   - 当前日期显示（“今天 + 月日 + 星期”）自动计算。
   - 顶部搜索框（PC 显示）。
   - 右侧：通知图标 + 用户头像下拉（个人中心、设置、退出登录）。

2. **左侧任务分类侧边栏**（PC 显示）
   - 快速添加任务输入框 + 添加按钮（当前使用弹窗提示，后续可写入列表）。
   - 主要分类列表：
     - 所有任务 / 我的一天 / 重要任务 / 已计划 / 已分配给我（每项显示数量）。
   - 标签列表：
     - #工作 / #学习 / #生活 / #健康（显示数量）。
   - “添加新标签”按钮（当前弹窗输入标签名）。

3. **主内容区**
   - 快捷操作栏：
     - 批量完成、批量删除（当前主要为 UI）。
     - 排序/筛选按钮 + 下拉菜单：按时间 / 优先级 / 标签。
     - 视图切换按钮（列表 / 卡片）。
   - 任务分组展示：
     - 今日待办 / 明日待办 / 未来7天 / 以后。
     - 每个任务卡片包含：
       - 复选框（完成状态）。
       - 标题、标签（工作/学习/生活/健康）。
       - 截止时间，子任务进度（如 2/5），预估番茄时长等信息。
       - 操作按钮：分享 / 复制 / 删除等（有的卡片仅有“更多”按钮）。
   - 交互要求：
     - 点击任务卡片（非复选框/操作按钮区域）在桌面端右侧打开任务详情；移动端提示“将打开详情页”（当前为 alert）。
     - 勾选复选框时，标题加删除线，并更新整体统计（当前为 UI 示例）。

4. **任务详情/编辑侧栏**
   - 可编辑属性：
     - 任务标题。
     - 描述文本域。
     - 截止日期时间（`datetime-local`）。
     - 重复设置（不重复/每天/每周/每月/自定义，含每周多选与结束日期）。
     - 提醒设置（准时、X分钟前、X小时前、1天前、自定义）。
     - 番茄专注设置：番茄数量、开始专注按钮。
     - 子任务列表（勾选、编辑、删除、新增）。
     - 标签管理（添加/移除）。
     - 任务移动到分类和标签。
   - 底部操作：取消 / 保存任务（当前本地 UI，不落地存储）。

5. **底部 Tab 导航**
   - Tab 项：待办 / 日历 / 番茄钟 / 数据周报 / 打卡 / 个人中心。
   - 本页高亮“待办”，其余 Tab 可点击跳到对应页面。

#### 3.3 日历视图页（`日历页面.html`）

- **核心目标**：以日历形式查看任务安排和专注计划，支持月 / 周 / 日视图切换与任务排程。  
- **主要模块**：

1. **顶部导航**
   - 返回按钮 + 标题“日历视图”。
   - “今天”按钮：回到当前日期视图（当前 UI 为主，逻辑可扩展）。
   - 当前日期显示。
   - 视图切换按钮：月视图 / 周视图 / 日视图（有激活态）。
   - 添加任务按钮（UI 占位，未来可对接创建任务流程）。
   - 用户头像下拉菜单同其他页面。

2. **主内容区**
   - 快捷操作栏（批量完成/删除、排序/筛选、视图切换）。
   - 日历视图区：
     - **月视图**：
       - 显示当月日期网格，周一到周日。
       - 每个日期格子可显示任务类型颜色点（工作/学习/生活/健康）。
     - **周视图**：
       - 以一周 7 天为列，展示每天的时间段任务条（如项目会议、学习前端等）。
     - **日视图**：
       - 小时刻度纵向列表 + 时间块任务条（如 9:00-10:00 会议）。
   - 待办任务池：
     - 列表形式显示“待排程任务”，支持拖动。
     - 拖拽到日历格子上（目前 console 打日志，可扩展为真实落地）。

3. **任务详情侧栏 & 专注历史侧栏**
   - 右侧同待办页结构，有任务详情编辑能力。
   - 另一个侧栏展示最近完成任务和“今日专注时段”示意条形图（静态）。

4. **交互要求**
   - 视图切换按钮切换不同 `.calendar-view` 容器的显示。
   - 拖拽 `.task-draggable` 到日历单元格可触发逻辑钩子（未来接入数据层）。

5. **底部 Tab**  
  同上，本页高亮“日历”。

---

#### 3.4 番茄专注页（`番茄钟页面.html`）

- **核心目标**：提供番茄钟计时与模式管理，并与任务和专注记录联动（当前主要为前端演示）。  
- **主要模块**：

1. **顶部导航**
   - 返回到待办页。
   - 标题“番茄专注”+ 副标题说明番茄工作法。
   - 中部显示当前模式（专注 / 短休息 / 长休息）与当前日期。
   - 右侧“本周专注统计”按钮（跳转到数据周报页）+ 用户头像菜单。

2. **左侧任务选择区**
   - 示例任务列表（完成项目提案 PPT / 学习 React Hooks / 健身锻炼）。
   - 每条任务显示标题、标签和预估番茄数。
   - 主要用于帮助用户选择当前专注关联的任务（当前无数据绑定，仅文案）。

3. **中间主计时器**
   - 模式切换按钮：
     - 专注 25' / 短休 5' / 长休 15'，点击切换模式与时长。
   - 圆形进度计时器：
     - 居中显示 `MM:SS`。
     - 外环进度条根据剩余时间动态变化。
   - 任务选择下拉框：选择本次番茄关联任务（也可不选，仅计时）。
   - 控制按钮：
     - 开始 / 暂停按钮（文字和图标状态切换）。
     - 重置按钮：回到当前模式的默认时长。
   - 当前会话统计：
     - 今日番茄数。
     - 已完成番茄循环数（如每 4 个一轮）。
     - 当前番茄序号。

4. **右侧提示与概要**
   - 专注小贴士列表。
   - 今日概要（预计今日番茄数、建议最晚结束时间、当前循环轮次等演示数据）。

5. **会话历史**
   - 列表展示本次页面打开以来完成的番茄记录（时间 + 关联任务）。
   - 支持“清空记录”。

6. **底部 Tab**  
  本页高亮“番茄钟”。

---

#### 3.5 数据周报页（`数据周报页面.html`）

- **核心目标**：以多种图表展示最近一周/一段时间内的任务完成和专注表现，为效率复盘提供参考。  
- **主要模块**：

1. **顶部导航**
   - 返回按钮 + 标题“数据周报”。
   - 当前日期显示 + 时间范围选择器（本周/上周/本月/自定义）。
   - 导出报告按钮（占位，后续可导出 PDF/图片）。
   - 分享按钮（当前 alert 提示）。
   - 用户头像菜单。

2. **概览卡片**
   - 本周完成任务总数（含较上周变化百分比）。
   - 本周专注总时长。
   - 本周完成番茄总数。
   - 本周最长连续专注番茄数。
   - 综合效率评分 + 等级（如 85 分 A 级）。

3. **图表区域（ECharts）**
   - 任务完成趋势图（折线图）：X 轴为周一~周日，Y 轴为任务数。
   - 任务分类统计（环形饼图）：工作/学习/生活/健康比例。
   - 番茄专注分析（柱状图）：各日番茄数量。
   - 专注时间段分布（热力图）：一周内各时间段专注密度。

4. **效率时段分析**
   - 以卡片形式展示不同时间段（清晨/上午/中午/下午/晚上/深夜）的完成任务数，并标注“最佳效率时段”。

5. **智能建议**
   - 根据示例数据给出自然语言建议，如：
     - 哪几天效率最高，建议安排重要任务。
     - 哪些时间段专注力下降，可安排轻任务。
     - 哪类任务更容易延期，建议拆分。
     - 番茄连续过多导致效率下降的提示。

6. **右侧专注历史侧栏**
   - 最近完成的几条专注记录。
   - 今日专注时段示意。

7. **底部 Tab**  
  本页高亮“数据周报”。

---

#### 3.6 打卡页（`打卡页面.html`）

- **核心目标**：支持每日打卡，记录习惯内容，展示日历与统计。  
- **主要模块**：
  - 今日打卡卡片：选择类型（学习/运动/早起/阅读等）、填写备注、点击“今日打卡”，更新状态。
  - 本月打卡日历：按月展示，过往日期标记已打卡，今天特殊高亮。
  - 最近打卡记录列表：本次会话内新增（前端内存级），可清空列表。
  - 打卡统计概要：本月打卡天数/当前连续打卡/最长连续打卡/本周打卡次数。
  - 打卡类型分布：用进度条展示不同类型占比。
  - 智能建议：关于如何稳定打卡和养成习惯的提示文案。
  - 顶部显示连续打卡天数和历史最佳记录。

---

#### 3.7 个人中心页（`个人中心页面.html`）

- **核心目标**：集中管理个人信息、账号安全、应用设置、数据隐私、成就与帮助信息。  
- **主要模块**：
  - 个人信息卡片（头像更换、昵称编辑、邮箱/手机号状态、等级与经验进度）。
  - 数据概要卡片（累计任务、累计专注时长、最长连续打卡、总番茄数）。
  - 存储管理（附件大小、图片/文档数量、清理缓存按钮）。
  - 帮助与支持：新手指南、常见问题、意见反馈、联系客服、评分入口。
  - 关于我们：版本号、用户协议、隐私政策等链接。
  - 账号安全：修改密码、绑定手机、两步验证、第三方账号绑定。
  - 应用设置：主题、通知、专注参数（番茄/休息时长、自动开始、屏蔽通知）、日历设置（周起始日、默认视图、农历）、数据隐私（导出、备份、清除、匿名统计）。
  - 退出登录、注销账号。

### 四、非功能性需求（简要）

- **一致的视觉风格**：
  - 使用统一 Tailwind 配色与阴影；字体大小整体偏紧凑，与个人中心/打卡页统一。
- **响应式布局**：
  - PC 端显示更多侧边栏与统计信息；移动端隐藏部分不关键区域，保持主要功能可用。
- **性能与体验**：
  - 不依赖后端即可本地演示交互；未来易于接入 API。
- **可扩展性**：
  - 各页面结构清晰、模块划分明确，便于重构为单页应用或接入路由框架。
